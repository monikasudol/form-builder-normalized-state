# Form builder

**Version 1.0.0**

Application which let you create input form


---

## Getting start

Clone this repo

```
https://github.com/monikasudol/form-builder.git
```

Write in the console

```
npm install
```

And then:

```
npm start
```

Use application at `http://localhost:3000/`
